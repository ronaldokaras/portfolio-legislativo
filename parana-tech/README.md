# Paraná Tech

**Projeto de Lei - Formação Gratuita em Programação e Tecnologia**

**Ementa:** Institui o Programa Paraná Tech – Formação Gratuita em Programação e Tecnologia para Jovens Paranaenses.

### 📄 Arquivos
- (./Projeto-Parana-Tech.pdf)

### Resumo
Programa estadual permanente com processo seletivo rigoroso, parcerias público-privadas e foco em jovens da rede pública. Busca reduzir a carência de profissionais qualificados em programação, IA, cibersegurança e análise de dados no Paraná.
